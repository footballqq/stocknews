# 能源战争三十六计：你的电动车，我的新武器

你有没有想过，那个让你在城市里自由穿梭，既环保又省钱的电动车，可能正悄悄地把你卷入一场你看不见的战争？

这不是危言耸听，这甚至不是未来，而是正在发生的现实。

故事要从半个世纪前说起。

1973年，中东的石油大佬们轻轻地关了一下阀门，整个西方世界就陷入了恐慌。加油站前排起长龙，物价飞涨，经济停滞。那一次，世界第一次见识到了**能源武器的恐怖威力**。那是一种简单粗暴的物理攻击，直接作用于你的油箱。

但那之后，世界似乎学乖了。大家开始搞能源储备，发展全球市场，建立规则。在很长一段时间里，我们都以为“石油武器”已经是一件过时的古董，被锁在了历史的陈列柜里。我们天真地以为，全球化和自由市场会像一个无形的大手，抚平所有的皱褶。

直到2022年，普京在乌克兰边境点燃战火，然后，他对着欧洲，也轻轻地拧了一下天然气的阀门。

历史，这个爱开玩笑的老头，又一次把我们拉回了那个熟悉的噩梦。欧洲的冬天变得异常寒冷，工厂停工，电价飙升。我们猛然发现，那个被遗忘的“能源武器”，不但没有生锈，反而**进化出了新的形态，变得更加致命**。

而这一次，战场已经不仅仅是中东的沙漠和西伯利亚的冻土。新的战场，可能就在你的车库里，在你手机的电池里，在你头顶的光伏板里。

欢迎来到，能源武器化的新时代。

## 从“石油美元”到“矿物人民币”

过去五十年，世界能源格局的主角是石油。谁控制了石油，谁就控制了世界经济的命脉。美国正是凭借着“石油-美元”体系，稳坐全球霸主的宝座。产油国负责挖油，美国负责印钱，大家一起玩一个叫做“全球化”的游戏。

这个游戏看起来很美好，直到有人开始不想玩了。

当世界主要国家都把宝押在“碳中和”上，以为从烧油烧气的旧时代，迈入用电用光的“美丽新世界”，就可以摆脱地缘政治的肮脏博弈时，他们忽略了一个致命的问题：

**电，从哪里来？**

风能、太阳能、电动车……这些听起来无比美好的词汇背后，指向的是同一个东西：**矿物**。

锂、钴、镍、石墨，还有那些名字古怪的稀土元素，它们才是清洁能源时代真正的“石油”。没有它们，你的特斯拉就是一堆废铁，风力发电机就是几个巨大的电风扇，光伏板就是一堆漂亮的玻璃。

而在这个新的赛道上，游戏规则的制定者，已经悄然换了人。

我们来看一组让西方世界夜不能寐的数据：

| 关键矿物/产品 | 中国全球产能占比 |
| :--- | :--- |
| 稀土 | 70%+ （精炼） |
| 石墨 | 70%+ |
| 锂 | 60%+ （精炼） |
| 钴 | 70%+ （精炼） |
| 太阳能电池板 | 80%+ |
| 电池（阳极） | 95% |

看明白了吗？

如果说上一个时代是“石油美元”的时代，那么新的时代，**一个“矿物人民币”的雏形正在浮现**。

中国几乎垄断了从矿物开采、提炼，到电池、太阳能电池板制造的整个产业链。这种控制力，比当年OPEC对石油的控制力还要可怕。

OPEC的武器是“断供”，让你没油可用。而中国的武器库里，工具要多得多。

它可以选择“断供”。2010年，因为钓鱼岛争端，中国暂停对日本出口稀土，日本的高科技产业立刻休克。2024年底，当中美科技战愈演愈烈时，中国限制了石墨和特定稀土的出口，直接导致福特汽车的部分工厂因为缺少来自中国的稀土磁铁而停产。

它也可以选择“淹没”。通过大规模生产和补贴，用洪水般的廉价产品冲垮你的本土产业，让你在不知不觉中，彻底失去自己“造血”的能力。等你反应过来，已经离不开它了。

这是一种更高级的玩法。它不再是直接攻击你的能源消耗端，而是**卡住你能源生产端的脖子**。你不是要搞绿色转型吗？可以，但造风车、造光伏板、造电池的原材料和设备，都得从我这里买。

这种依赖，比石油依赖更加致命。因为石油是消耗品，你今天不用，明天还得买。而风电站、光伏板是基础设施，一旦建成了，就会用上二三十年。这意味着，你的整个能源体系，将在未来几十年内，都建立在另一个国家的掌控之上。

这已经不是简单的商业竞争，这是**国家层面的阳谋**。

## 美国的“左右互搏”与中国的“降维打击”

面对这个全新的局面，曾经的霸主美国，显得有些手足无措。

它一方面享受着页岩油革命带来的“能源独立”的幻觉。今天的美国，确实产的能源比用的多。但“能源独立”是个彻头彻尾的伪命题。因为油价是全球定价的，只要你还在这个市场里，任何地方的风吹草动，都会影响到你家门口加油站的价格牌。

更重要的是，当美国还在为多挖几桶油而沾沾自喜时，它惊恐地发现，在通往未来的新能源赛道上，自己已经被远远甩在了身后。

于是，我们看到了一个非常拧巴的美国。

特朗普政府和拜登政府，看似政策南辕北辙，但在“能源武器”这个问题上，却出奇地一致。他们都在用关税、制裁等手段，试图延缓中国的脚步，同时又不得不投入巨资，补贴本土的清洁能源产业，希望能够“亡羊补牢”。

但这很难。

因为中国玩的，是一种“降维打击”。

当你还在纠结于意识形态，纠结于是该用“市场”还是“计划”时，中国已经利用自己强大的国家能力和制造业优势，把整个产业链做成了一个成本的“黑洞”。任何试图挑战它地位的玩家，都会被这个黑洞无情地吞噬。

前美国参议员乔·曼钦的一句话，道出了西方的绝望：“我们难道要用一个不可靠的外国供应链，去取代另一个不可靠的foreign supply chain吗？”

这话说对了一半。

石油和天然气供应链的风险，和清洁能源供应链的风险，完全不是一个量级。

**普京关掉天然气，你家冬天可能真的会断暖。但如果中国限制电池出口，你的电动车最多是晚几个月提车，或者价格贵一点。**

前者是“要不要活”的问题，后者是“活得好不好”的问题。

中国掐断你的清洁能源供应链，不会让你的国家立刻停摆，但它会让你痛苦，会延缓你的转型进程，会让你在未来的竞争中，因为高昂的能源成本而失去优势。

这是一种温水煮青蛙式的绞杀。

更可怕的是，电网本身，也成了新的战场。美国FBI已经多次警告，来自中国的黑客已经渗透到美国关键的基础设施网络中，尤其是电网。在用电高峰期，或者在关键时刻，一次精准的网络攻击，就可能造成远超传统战争的破坏。

## 唯一的出路：一场“新军备竞赛”

面对这样一个复杂而危险的局面，我们该怎么办？

答案可能有些反直觉。

唯一的出路，可能不是退缩，不是去搞什么“脱钩断链”，而是**更疯狂地投入到这场新的能源竞赛中**。

这篇文章的核心观点，也是最令人振奋的一点是：**对能源武器的恐惧，将成为推动清洁能源发展的最强动力，甚至超过对气候变化的担忧。**

这听起来很讽刺，但却无比现实。

几十年来，我们呼吁环保，呼吁为了地球、为了子孙后代而转向清洁能源。但这种基于道德的呼吁，在现实的国家利益面前，往往显得苍白无力。

但现在，情况变了。

发展清洁能源，不再仅仅是一个环保问题，它成了一个**国家安全问题**。

当一个国家意识到，自己的经济命脉、军事命脉，都可能因为另一个国家“不高兴”而被切断时，它会不惜一切代价去摆脱这种依赖。

这将在全球范围内，掀起一场新的“军备竞赛”。

只不过，这次竞赛的标的，不再是航母和核弹，而是**太阳能电池板的转化效率、电池的能量密度、电网的智能化水平，以及对关键矿产资源的控制**。

对于美国和欧洲来说，他们必须重新审视自己的战略。仅仅依靠市场和私营企业，已经无法赢得这场竞赛。他们需要更大规模的国家干预，需要像当年搞“阿波罗计划”一样，将清洁能源和能源安全，提升到国家战略的最高等级。

即使这意味着更高的成本，也在所不惜。因为“安全”的溢价，远高于“便宜”的诱惑。

而对于我们普通人来说，这场竞赛，既是挑战，也是机遇。

挑战在于，我们可能会在未来一段时间内，面临能源价格的上涨和供应链的不稳定。

机遇在于，这场由地缘政治驱动的能源革命，将以前所未有的速度和规模，创造出全新的产业、全新的技术和全新的商业模式。

从长远来看，一个更加多元化、更加本地化的能源体系，对所有人来说，都是一件好事。它会让我们更少地受到地缘政治的勒索，更多地掌握自己的命运。

所以，下次当你开着你的电动车，享受着片刻的宁静时，不妨想一想背后这场波澜壮阔的全球博弈。

你不是在简单地开车，你是在用自己的方式，为这场决定未来一百年国运的战争，投下自己的一票。

而这场战争的结局，无人知晓。但唯一可以确定的是，**那个以为靠着几桶石油就能号令天下的时代，已经一去不复返了。**

新的牌局，已经开始。
