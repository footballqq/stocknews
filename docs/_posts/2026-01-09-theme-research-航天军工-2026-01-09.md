## 详细部分

### 1. 事件或主题概述

近期航天军工领域迎来多重催化。一方面，SpaceX CEO埃隆·马斯克在社交媒体确认，公司最终目标是每年生产多达1万艘星舰飞船，远超此前每年1000艘的规划。为此，SpaceX已在得克萨斯州星舰基地投资2.5亿美元建设GigaBay工厂，占地70万平方英尺，设计年产能达1000枚火箭。另一方面，美国前总统特朗普宣布计划将2027财年国防预算提升至1.5万亿美元，较当前水平增长超过50%，旨在打造“梦想军队”。

受此推动，全球资本市场反应积极。美股航天国防股指数ETF（ARKX）盘中大涨超4%，Kratos、Aerovironment、洛克希德·马丁等军工巨头股价显著上涨。A股市场中，商业航天概念持续发酵，多个细分方向表现亮眼，尤其以“有源相控阵天线”为代表的高技术壁垒环节受到重点关注。

### 2. 核心驱动与影响分析：投资逻辑分步解释

#### 驱动因素一：SpaceX产能扩张重塑行业格局
- **逻辑链**：星舰作为可重复使用重型运载工具，其大规模量产将显著降低发射成本 → 推动低轨星座部署加速 → 带动整个商业航天产业链需求爆发
- **影响范围**：火箭制造、卫星生产、地面设备、材料供应等全链条

#### 驱动因素二：美国军费大幅增长确定性强
- **逻辑链**：地缘政治紧张 + 战略竞争升级 → 国防预算刚性增长 → 军工企业订单与业绩确定性提升
- **数据支撑**：2026年美国军费预计维持高位，2027年若实现1.5万亿美元目标，将开启新一轮扩张周期

#### 驱动因素三：商业航天进入规模化应用阶段
- **逻辑链**：Starlink成功验证商业模式 → 全球多国启动低轨星座计划 → 卫星制造从“小批量定制”转向“批量化生产”
- **技术拐点**：有源相控阵天线成为标配，T/R组件需求激增

#### 驱动因素四：国产替代与自主可控加速
- **逻辑链**：美国技术封锁加剧 → 中国加大自主研发投入 → 高壁垒环节迎来“进口替代+增量需求”双重红利
- **重点领域**：GaN半导体、T/R组件、星载相控阵天线系统

### 4. 主要风险与不确定性

1. **政策执行风险**：特朗普军费提案需国会批准，存在不确定性；中美科技脱钩可能加剧供应链扰动
2. **技术迭代风险**：航天技术发展迅速，现有技术路线存在被颠覆可能
3. **产能释放不及预期**：星舰量产涉及复杂工程体系，实际达产节奏可能慢于规划
4. **估值回调风险**：部分概念股涨幅较大，存在短期回调压力
5. **国际局势变化**：地缘冲突升级可能推高融资成本，抑制风险资产表现
