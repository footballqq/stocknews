# 债市开放与供给扩容双轮驱动 —— 国债市场研究摘要

## 简要部分

**核心观点：** 国债市场在外资准入扩容与地方债供给双轮驱动下，短期波动不改中长期开放与扩容趋势。当前收益率已具备配置价值，但需关注供给节奏与政策落地节奏。

1. 摩根大通与星展银行获中国非金融企业债务融资工具一般主承销商资质，外资承销深度拓展。
2. 10年期国债活跃券收益率下行0.95bp至1.889%，30年期下行2.1bp至2.314%，债市短期修复。
3. 央行单日净投放99亿元7天逆回购，同时开展11000亿元买断式逆回购，资金面维持宽松。
4. 外资承销机构扩容至14家，覆盖美、法、英、日、新及中港澳台地区，债市制度型开放深化。
5. 2026年新增地方债发行规模预计扩容，专项债发行节奏将较2025年进一步前置。
6. 置换隐债专项债预计前置发行，特殊新增专项债于二三季度接续发力，错峰特征明显。

9. 一级存单3M国股需求较好，1Y期报价1.64%-1.66%，负债端成本压力边际缓和。
10. 机构提示反弹不宜过度乐观，需关注经济数据与供给节奏的边际变化。

## 详细部分

### 1. 事件或主题概述

近期国债市场呈现“外资开放深化 + 供给节奏变化 + 短期波动修复”三大特征。一方面，外资机构持续扩容，摩根大通与星展银行获非金融企业债务融资工具主承销商资质，标志着中国债市制度型开放进入新阶段；另一方面，地方债发行节奏与规模预期调整，中信证券预计2026年新增地方债供给将扩容且前置发力。市场层面，10年与30年期国债收益率在连跌后出现技术性修复，资金面维持宽松，但机构对反弹持续性保持谨慎。

### 2. 核心驱动与影响分析：投资逻辑分步解释

#### 驱动一：外资准入扩容，提升市场深度与定价效率
- **逻辑链：** 资质获批 → 外资承销范围扩大 → 引导跨境资本流入 → 增强人民币资产配置需求
- **影响：** 外资机构凭借全球网络与投资者基础，可更高效连接境内外市场，提升国债与信用债的国际认可度，长期利好收益率曲线稳定。

#### 驱动二：地方债供给扩容预期，短端承压但中长期利好流动性
- **逻辑链：** 化债资金前置 + 专项债扩容 → 供给节奏前倾 → 短期资金面承压 → 倒逼货币宽松配合
- **影响：** 供给增加短期或压制价格，但为配合发行，央行或维持宽松流动性，中长期利好债市环境。

#### 驱动三：技术性修复与配置价值显现
- **逻辑链：** 收益率下行 + 资金宽松 → 配置价值提升 → 机构逢低吸纳 → 市场企稳
- **影响：** 1.89%的10年期国债已具备吸引力，尤其对保险、银行等负债端稳定的机构而言。

### 3. 相关ETF或行业的潜在影响

- **国债ETF（如：国债ETF、10年期国债ETF）：** 短期受益于收益率下行带来的价格反弹，中长期受益于外资流入与市场深度提升。
- **政金债ETF：** 同样具备配置价值，且流动性优于部分地方债。
- **银行与券商板块：** 外资承销资质扩容利好具备跨境能力的头部券商（如 中信证券 、 中金公司 ），银行间市场活跃度提升亦有助于银行债券交易与承销业务。
- **信用债ETF：** 外资参与度提升有助于信用债市场定价效率改善，但需关注资质下沉风险。

### 4. 主要风险与不确定性

- **供给节奏超预期：** 若地方债发行节奏快于市场承接能力，可能引发利率上行压力。
- **政策落地不及预期：** 化债与特殊再融资债发行进度若滞后，可能削弱流动性宽松预期。
- **海外利率波动：** 美债收益率若大幅上行，可能通过利差收窄压力传导至国内债市。
- **经济数据回暖：** 若基本面改善超预期，可能削弱债市避险需求。
