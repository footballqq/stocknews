# Coinbase CEO 上演“骚操作”，一句话引爆8万美金赌局，预测市场要玩脱了？

故事要从一场本该严肃无聊的财报电话会议说起。

主角是布莱恩·阿姆斯特朗（Brian Armstrong），美国最大加密货币交易所Coinbase的CEO。他身后的这家公司，刚刚经历了又一个季度的市场搏杀，正准备向华尔街的分析师们汇报战果。

电话会议的空气里，通常都弥漫着金钱、数据和预期的味道。分析师们像一群嗷嗷待哺的雏鸟，等着CEO吐出关于利润、增长和未来指引的“干货”。

会议进行得很顺利，一切都按部就班。直到最后，当所有人都以为可以收拾东西下班时，阿姆斯特朗却突然话锋一转，上演了一出让所有人目瞪口呆的“骚操作”。

他清了清嗓子，对着电话那头的整个华尔街说：

> “我有点分心了，因为我一直在关注一个关于Coinbase今天会说什么的预测市场。”

话音未落，听众们可能还在纳闷，这是什么新式凡尔赛吗？

紧接着，阿姆斯特朗像一个脱口秀演员在结尾“call back”自己的梗一样，慢悠悠地念出了一串词：

> “我就是想在这里补充一下：**比特币、以太坊、区块链、质押（staking）和 Web3**——确保我们在电话会议结束前把这些词都说全了。”

说完，他潇洒地结束了会议。

留下的，是电话另一端一脸错愕的分析师，和在预测市场上炸开了锅的赌徒们。

**就因为他这轻描淡写的一句话，一个不大不小，总金额8.4万美金的赌局，瞬间被引爆。**

这到底是怎么回事？

**1. CEO的“上帝之手”**

阿姆斯特朗提到的“预测市场”，是一种越来越火的新兴事物。你可以把它理解成一个“万物皆可赌”的平台。

传统的赌局，赌球、赌马、赌选举结果。而**在预测市场上，你可以赌任何未来可能发生的、并且结果可以被公开验证的事件。**

比如，美联储下个月会不会降息？苹果下一代iPhone会不会上新功能？甚至，某个明星会不会在社交媒体上发表某个言论？

这次，在Kalshi和Polymarket这两个预测市场平台上，就有人开了一个局：**赌Coinbase的高管们会不会在这次财报电话会议上提到某些特定的“行业热词”。**

这些词包括“稳定币”（stablecoin）、“机构”（institution）、“保证金”（margin）等等。当然，也包括阿姆斯特朗最后念出的那几个。

这个赌局吸引了大约8.4万美金的资金。

你可以想象一下这个场景：

一群交易员，像盯着股票K线一样，死死盯着Coinbase财报会议的直播。手里拿着一张“词汇表”，每当有高管说出一个列表上的词，他们就在心里打一个勾。

会议快结束了，分析师的提问环节也走到了尽头。眼看着“比特币”、“以-太坊”这几个关键的词还没被提及。那些押注“会说”的交易员，心已经提到了嗓子眼。

就在这千钧一发之际，CEO本人，像一个仁慈的上帝，亲自下场，轻飘飘地念出了“咒语”。

**那些即将归零的合约，瞬间“起死回生”，变成了真金白银。**

看着屏幕上瞬间翻转的赔率，交易员们的心情，恐怕比坐过山车还刺激。

事后，阿姆斯特朗还在社交媒体X上发了条帖子，轻描淡写地说：

> “哈哈哈，这太好玩了。”

他解释说，这是一时兴起。当时公司内部聊天群里有人发了那个预测市场的链接，他看到了，觉得有趣，就在会议的最后“玩”了一下。

**但这件事，真的只是“好玩”而已吗？**

**2. “预测未来”还是“操纵未来”？**

阿姆斯特朗的这个“玩笑”，像一把锋利的刀，划开了预测市场光鲜的外衣，露出了里面令人不安的内核。

这件事最核心的争议点在于：**它以一种极其直白的方式，暴露了预测市场是多么容易被它们本应观察的对象所操纵。**

预测市场的支持者们，一直以来都给它描绘了一个非常美好的蓝图。他们认为，这种市场是“集体智慧”的体现，是一种比民意调查和专家预测更牛逼的“预言机器”。

理论上，当成千上万的人用真金白银对未来下注时，市场的价格就能汇集所有可用的信息，从而给出一个最接近真相的概率。

听起来很美好，对吧？

但Coinbase这件事，就像一记响亮的耳光，打在了“集体智慧”的脸上。

**所谓的“集体智慧”，在CEO的“上帝之手”面前，显得如此不堪一击。**

他甚至不需要任何复杂的布局，不需要调动任何资源，只需要动动嘴皮子，就能让市场的结果随心所欲地改变。

这已经不是在“预测”未来了，这简直是在“创造”未来。

Coinbase的发言人后来出来打圆场，说CEO的言论是“轻松的、随口的”，只是引用了网上的讨论。并且强调，公司内部有规定，禁止员工参与和公司相关的预测市场。

Kalshi和Polymarket这两个平台，也识趣地拒绝发表评论。

但所有人都心知肚明，潘多拉的魔盒一旦打开，就很难再关上了。

美国商品期货交易委员会（CFTC）的指导方针里，明确规定了受监管的平台，不应该上架那些“容易被操纵”的合约。

现在，“词汇提及市场”（Mention Markets）这个奇葩的产物，正在疯狂试探这条监管的红线。

一位律师评论说：

> “我们首先需要回答一个核心问题：这里真的有什么值得监管的东西吗？还是说，这只是一个理论上听起来不错，但实践中行不通，然后大家转向下一个新事物的昙花一现的功能？”

这个问题，现在没有人能给出答案。

**3. 赌场还是价值投资？**

更有意思的是，Coinbase自己，就是预测市场这个赛道上的“玩家”。

**它同时持有Kalshi和Polymarket的股份。**

就在这次引发争议的财报电话会议上，阿姆斯特朗还雄心勃勃地表示，公司计划扩展到“事件合约”领域，致力于成为一个金融产品的“万能交易所”（Everything Exchange）。

一边是规则的制定者，一边是游戏的参与者，现在又亲自下场当了一回“裁判”。Coinbase这波操作，真是把“既要...又要...”玩明白了。

网上对这件事的反应，也是一半玩笑，一半忧虑。

Polymarket的官方X账号，用“恶魔般的杰作”（diabolical work）来形容阿姆斯特朗的行为。

一位体育预测市场平台的交易主管则忧心忡忡地说：

> “如果CFTC说这事太离谱了，那整个行业可能都要玩完。”

“词汇提及市场”虽然只是预测市场里一个非常小众的角落，但它就像一个放大镜，照出了整个行业存在的种种“怪现状”。

交易员们为了“扣字眼”，吵得不可开交。

*   一个词的复数形式算不算？（比如赌“immigrant”，说“immigrants”算吗？）
*   一个词出现在复合词里算不算？（比如赌“ICE”，说“ice water”算吗？）

根据Kalshi的奇葩规则，“immigrants”可以满足“immigrant”的赌约，但“immigration”就不行。而“ice water”真的可以满足“ICE”的赌约。

**这种“抠字眼”式的赌博，与其说是“信息聚合”，不如说是一场文字游戏。**

阿姆斯特朗那句半开玩笑的话，就像一面镜子。

它照出的，不是什么“集体智慧”的光辉，而是人群是多么容易被玩弄于股掌之间。

| 预测市场的理想与现实 | |
| :--- | :--- |
| **理想 (The Dream)** | **现实 (The Reality)** |
| 汇聚集体智慧，预测未来 | 容易被少数人操纵 |
| 发现价格，对冲风险 | 沦为“抠字眼”的文字游戏 |
| 更高效的信息市场 | 监管模糊，充满争议 |
| 去中心化的决策工具 | 权力依然集中在“上帝”手中 |

**4. 加密世界的“成人礼”**

从更宏大的视角来看，Coinbase CEO的这次“行为艺术”，或许是整个加密世界正在经历的一场“成人礼”。

多年以来，加密世界一直以一种“屠龙少年”的形象示人。它反叛、去中心化、高举着“代码即法律”（Code is Law）的大旗，试图颠覆传统金融的“旧世界”。

但随着比特币ETF的通过，华尔街的巨头们纷纷入场，加密世界正在不可避免地被“招安”，被纳入现有的金融监管体系。

这个过程中，必然会充满各种光怪陆离的冲突。

**一边是加密原教旨主义者对“去中心化”理想的坚守，另一边是资本和权力在现实世界中的无情碾压。**

预测市场，正是这种冲突的绝佳体现。

它诞生于加密社区，身上流淌着“去中心化”的血液。但当它试图进入主流，就必须戴上监管的“紧箍咒”，必须面对像阿姆斯特朗这样，可以凭借个人影响力“一言兴邦，一言丧邦”的“国王”。

这次事件，就像一个寓言。

它告诉我们，**无论技术的外衣多么花哨，人性的贪婪、权力的诱惑、以及操纵与被操纵的游戏，永远不会缺席。**

阿姆斯特朗的“骚操作”，究竟是一次无伤大雅的玩笑，还是一次对市场规则的公然挑衅？

预测市场的未来，是会成为一个真正发现价值的工具，还是会沦为一个更加疯狂、更加难以监管的“大赌场”？

没有人知道答案。

但可以肯定的是，当Coinbase的CEO开始在财报会议上，像一个赌场荷官一样，公开“摇骰子”的时候，这个行业，就已经回不去了。

那句“lol this was fun”，充满了后现代的解构意味。

**在绝对的权力面前，所谓的规则、智慧和预测，都显得有些可笑。**

这或许，才是这场8.4万美金赌局背后，最令人不寒而栗的真相。
