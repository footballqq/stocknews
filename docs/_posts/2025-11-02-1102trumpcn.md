# 川普王者归来，北京这次为什么不慌了？

2025年初，当那个熟悉的红领带身影再次踏入白宫时，大洋彼岸的北京，空气中弥漫着一种复杂的情绪。精英圈子里，人们私下议论着一场“前所未有的风暴”，四年前那场贸易战的硝烟与创伤，仿佛还未散尽。中国的经济引擎正踩着刹车，对美贸易的依赖像一根脆弱的神经，随时可能被再次挑断。所有人都屏住了呼吸，等待着另一只靴子落下。

然而，仅仅六个月，画风变得让人有些猝不及防。

那场预想中的狂风暴雨，来了，但似乎又没完全来。北京不仅稳稳接住了川普的“三板斧”，甚至还在你来我往的交锋中，意外地发现自己手里多了几张能让对方坐下来好好说话的王牌。曾经的焦虑与不安，正悄然被一种新的情绪所取代。在北京的政策研讨会上，一个短语开始被反复提及，带着一丝自嘲，更带着一种 newfound 的底气：

**“那又怎样？”**

贸易战来了？那又怎样？
川普要加压？那又怎样？

这种“气定神闲”从何而来？这半年里，中美这对世界上最复杂的对手之间，到底发生了什么？故事，远比想象的更精彩。

## “杀敌一千，自损一千二”的游戏

川普二进宫，果然没让世界“失望”。二月、三月，熟悉的关税大棒再次挥舞，理由是打击来自中国的芬太尼。四月，更大规模的“解放日”关税来袭。北京的反应也毫不含糊：你加我也加，你限制我，我也限制你。一套对等反制组合拳打出去，到四月中旬，美国对中国商品的关税税率飙GE到了惊人的150%。

空气紧张到了冰点，所有人都觉得，一场不死不休的贸易血战在所难免。中国的经济能扛得住吗？

**但接下来，有趣的一幕发生了。**

先坐不住的，竟然是挥舞着大棒的华盛顿。

原本以为会死扛到底的中国，展现出了惊人的韧性。反倒是美国那边，股市开始晃动，债券市场传来警报，零售行业叫苦不迭，消费者的账单也越来越难看。Adam Posen在《外交事务》里提到的那个观点，被活生生地验证了：对美国来说，关税带来的“供给冲击”远比对中国造成的“需求冲击”更难消化。

这是个很简单的道理。对美国而言，这是**“断供之痛”**。中国不卖了，货架上就真的空了，老百姓的生活成本立马飙升。而对中国来说，这是**“需求之痛”**。卖给美国的少了，短期内是难受，但船大掉头慢，广阔的国内市场可以消化一部分，更何况过去几年，中国早就没把鸡蛋放在一个篮子里。

当川普发现，自己的关税大棒打在中国身上，疼的却是自家后院的“票仓”时，他的声调开始变了。5月12日，中美在日内瓦签署协议，同意降低关税。6月5日，两国领导人通了电话。第一回合的极限施压，就这么戏剧性地结束了。

北京的决策者们惊讶地发现，原来自己比想象中更能“抗揍”，而对手，似乎比想象中更“怕疼”。

## 手里的“王炸”：稀土与产业链

如果说关税战的“反杀”让北京找回了些许自信，那另一张牌的威力，则彻底让天平发生了倾斜。

这张牌，就是**稀土**。

现代工业的“维生素”，从手机、电动车到F-35战斗机，哪哪都离不开它。而这张牌，几乎完全攥在中国手里。4月4日，中国宣布暂停关键稀土出口。这一招，精准地打在了美国的“七寸”上。华盛顿的汽车、飞机制造商们，一夜之间就有了断炊的风险。

在日内瓦的谈判桌上，稀土出口是核心议题之一。中国同意解禁，但留了个心眼，指责华盛顿违反协议，对华为等中企搞小动作，因此“延迟”了履行承诺。

这一“延迟”，再次让华盛顿暴跳如雷。他们对中国学生搞签证限制，对喷气发动机搞出口限制，用尽各种手段施压。但这种种过激反应，恰恰向北京证明了一件事：**稀土这张牌，我们打对了。** 它是如此重要，以至于华盛顿愿意为此付出巨大的外交代价。

这是中美建交以来，中国第一次在战略层面，找到了一个能有效“卡住”对方脖子的关键节点。

当然，仅仅一张“王炸”还不够。过去十年，尤其是川普第一任期之后，中国一直在默默做一件事：**经济的“去美国化”和“多元化”**。

到2024年底，欧盟和东盟已经取代美国，成为中国更大的贸易伙伴。更绝的是“转口贸易”的运用。大量中国商品，通过越南、墨西哥等第三国“换个马甲”，就绕过了美国的关税壁垒，进入美国市场。川普政府当然想打击这种做法，但问题是，除非美国愿意跟全世界所有和中国有贸易往来的国家都“脱钩”，否则这种“曲线救国”的贸易路线，几乎无法被彻底封死。

**“朋友圈”变大了，对单一客户的依赖变小了，还有了能卡住对方命脉的硬通货。** 这就是北京敢说“So what?”的底气所在。

## 终于等来了那个“商人”川普

北京的决策者们现在觉得，他们终于等来了2016年时所期待的那个“商人”川普。

和第一任期相比，川普2.0政府有个显著的变化：**意识形态色彩淡了，交易色彩浓了。**

“新冷战”的口号很少被提及，对中国政治体制的攻击也收敛了许多。甚至在台湾问题上，华盛顿也显得异常“克制”。川普不仅在7月底对台湾加征了20%的关税，还拒绝了赖清德“过境”纽约的请求，这打破了多年来的传统。

这种对政治议题的“不感兴趣”，和对贸易逆差的“极度痴迷”，恰恰是北京最希望看到的。一个只关心账本的对手，总比一个满脑子都是“主义”的对手，要好打交道得多。

| 川普政府对华策略对比 | 第一任期 (2017-2021) | 第二任期 (2025- ) |
| :--- | :--- | :--- |
| **核心团队** | 鹰派“新冷战”主义者 | 忠于川普本人的“生意人” |
| **主要议题** | 贸易、人权、意识形态、台湾 | **贸易逆差** (核心中的核心) |
| **对华态度** | 充满敌意的战略竞争 | 结果导向的交易对手 |
| **北京应对策略** | 试图管理其团队 (如通过库什纳) | **试图直接管理川普本人** |

在川普的第一个任期，北京曾试图通过管理他的团队（比如女婿库什纳）来影响决策，但收效甚微。这一次，北京意识到，最有效的方式是**“搞定川普本人”**。只要能让川普在贸易上获得他想要的“胜利”，其他问题都可以谈。

就连曾经以“反华先锋”著称的马可·卢比奥，在出任国务卿后，对中国的批评也变得异常谨慎。整个团队，都紧紧围绕着川普的指挥棒在转。

因此，获得与川普本人面对面交流的机会，变得至关重要。这就是为什么北京热切期盼今年秋天能在主场举办一次中美元首峰会。按照外交礼仪，习近平2023年访问了美国，这次轮到川普来访，这本身就象征着一种“平等”的姿态。

北京希望，能在这场峰会上，直接探一探川普的口风，看看能否达成某种“大交易”，用经济上的让步，换取政治上的空间。比如，让美国在对台军售上有所收敛，或者更明确地表态反对“台独”。这些让步，即便不能立刻解决台湾问题，也能极大地动摇岛内的“倚美”信心。

## 上半场得意，下半场焦虑

然而，贸易上的 empowered 感，并没有完全冲昏北京的头脑。他们清醒地认识到，这很可能只是“上半场”的暂时领先。

许多中国学者和决策者都担心，美国对贸易的关注只是战术性的、暂时的。一旦川普认为自己在贸易上取得了“胜利”，他很可能会腾出手来，转向更棘手的政治和安全问题。那才是北京真正感到“肉疼”的地方。

比如，重启对中共党员及其家属的签证限制，加大对台湾的军事和外交支持，或者像2020年那样，将矛头对准中国的政治体制，试图“将中共和中国人民分开”。

**贸易上的问题，是钱的问题，总有讨价还价的余地。而政治上的问题，是生存问题，几乎没有妥协空间。**

一旦华盛顿将竞争的焦点从“贸易逆差”转向“意识形态”，中美关系将再次滑向自由落体的深渊。

因此，北京现在采取的策略，可以概括为**“拖”**字诀。

他们愿意通过持续的、零敲碎打的贸易谈判，来不断地解决小的贸易摩擦。比如，承诺多买一些美国货，换取美国放松一些高科技产品的出口管制；甚至愿意搞一些“自愿出口限制”，少卖一些电池之类的产品到美国。

这样做的目的，不仅仅是为了解决贸易问题本身，更是为了**让贸易问题始终占据中美关系的C位**，让川普的注意力始终集中在“做生意”上，从而没有精力去开启其他更危险的“副本”。

同时，北京在任何潜在的协议中，都会埋下“退出条款”。如果川普变卦，或者华盛顿在台湾等问题上采取敌对行动，北京会毫不犹豫地撕毁协议，并用自己手中的经济牌进行报复。

## 结语：一场短暂的喘息

展望未来，北京的心态是矛盾的。

一方面，他们在与川普的第二轮交锋中，意外地占据了上风，手握稀土、产业链和庞大的市场，让他们有了前所未有的谈判底气。他们希望与这位“商人总统”进行面对面的谈判，用一份份贸易协议，锁定未来三年相对平稳的中美关系。

但另一方面，这种底气又是脆弱的。他们深知川普的不可预测性，也明白贸易上的胜利，可能只是为下一场更严峻的斗争，赢得了一段短暂的喘息时间。

北京的决策者们，就像一群走在钢丝上的杂技演员，他们小心翼翼地维持着平衡，试图在这场充满变数的博弈中，为自己争取到最有利的位置和最宝贵的时间。

无论他们现在感觉多么强大，有一点是确定的：**必须为可能到来的、更糟糕的局面，做好万全的准备。** 因为在这场世纪博弈中，真正的终局，还远未到来。
