# 市场研究摘要：2025年12月REITs市场结构性分化与2026年投资策略

## 简要部分

### 主要话题与市场趋势概述
2025年12月，中国REITs市场整体跑输大盘，呈现显著的结构性分化特征。**新型基础设施REITs表现亮眼，消费与仓储物流板块延续强势，而交通与能源类资产回调明显**。政策层面持续扩容底层资产范围，外资参与度提升，市场流动性改善明显。展望2026年，在“十五五”规划开局与稳增长政策发力背景下，REITs市场有望迎来估值修复与结构性机会并存的新阶段。

## 详细部分

### 1. 事件或主题概述

2025年12月，中国公募REITs市场在整体弱势中展现出鲜明的结构性特征。中证REITs指数月度跌幅达2.92%，弱于同期主要股指表现。从资产类别来看，**园区、仓储物流、保障性租赁住房及新型基础设施等与产业升级和民生保障相关的资产表现相对坚挺，而交通与能源类传统基建资产则面临一定压力**。

政策层面迎来重要突破：国家发改委发布《基础设施领域不动产投资信托基金（REITs）项目行业范围清单（2025年版）》，将发行范围从13个行业扩展至15大类，新增商业办公设施、城市更新设施，并明确数据中心、人工智能基础设施的独立分类地位。同时，外资机构首单仓储物流REITs成功上市，标志着中国REITs市场国际化进程迈出关键一步。

### 2. 核心驱动与影响分析：投资逻辑分步解释

#### 驱动因素一：政策持续扩容，底层资产质量优化
- **政策导向明确**：新版行业清单将高能级城市的优质商业办公、城市更新项目纳入试点，提升资产准入标准与长期价值。
- **数字经济赋能**：数据中心、AI基础设施独立分类，反映国家战略导向，增强相关资产的资本化能力。
- **影响路径**：政策红利 → 资产供给扩容 → 投资者选择丰富 → 定价效率提升 → 板块估值重构

#### 驱动因素二：市场分化加剧，资产质量成关键变量
- **微观运营差异显现**：同一类别下不同项目表现迥异，如交通类REITs中招商公路上涨1.10%而浙商沪杭甬下跌10.60%，反映市场对区域经济、路网竞争格局的精细化定价。
- **现金流稳定性溢价凸显**：保障性租赁住房、高标仓物流等具备稳定租约与政策支持的资产更受青睐。
- **影响路径**：市场成熟度提升 → 定价逻辑从“类别偏好”转向“个体价值” → 优质资产获得估值溢价

#### 驱动因素三：外资参与与流动性改善
- **安博REITs成功发行**：首单外资背景仓储物流REITs上市，验证国际资本对中国基础设施资产的认可。
- **流通盘占比达50%**：市场深度显著提升，降低交易摩擦，增强机构配置意愿。
- **影响路径**：外资进入 + 流动性改善 → 市场定价效率提升 → 吸引更多长期资金入场

#### 投资逻辑演进：从“红利资产”到“成长+稳定”双轮驱动
当前REITs投资逻辑正经历三重升级：
1. **从“高分红”转向“分红+估值弹性”并重**：消费与新基建类REITs不再仅依赖分红收益，更具备资产增值潜力；
2. **从“政策套利”转向“基本面驱动”**：市场更加关注底层资产的运营效率、区域经济支撑力与长期需求趋势；
3. **从“单一配置”转向“战略协同”**：REITs作为连接实体经济与资本市场的工具，正被纳入国家战略实施框架。

#### ETF产品影响
目前市场上已有**中银证券安泰REIT**、**建信中关村REIT**等多只公募REITs基金，未来随着资产类别丰富与发行常态化，预计将有更多**REITs指数基金与行业主题ETF**推出，进一步提升市场流动性与配置便利性。

### 4. 主要风险与不确定性

1. **宏观经济复苏不及预期**：若消费与工业生产恢复缓慢，可能影响商业、物流、产业园等类型REITs的租金收入与 occupancy 率。
2. **利率波动风险**：REITs作为固定收益类资产，对利率敏感。若货币政策转向紧缩或债市波动加剧，可能压制估值。
3. **资产估值偏高风险**：部分热门板块（如消费、新基建）已积累一定涨幅，存在短期回调压力。
4. **政策执行不确定性**：尽管政策方向明确，但具体项目审批节奏、税收优惠细则等仍存变数。
5. **运营管理风险**：REITs表现高度依赖底层资产的运营能力，管理人经验不足或区域竞争加剧可能影响实际分红。
