# 券商行业市场研究摘要（2024年11月）

## 简要部分

### 主要话题与市场趋势概述

当前中国券商行业正处于结构性复苏与战略转型的关键阶段。**A股再融资市场爆发式增长、居民“存款搬家”加速财富管理转型、头部券商并购整合推进，三大趋势共同构成券商板块的核心驱动力**。行业基本面改善明显，投行业务回暖带动业绩修复，财富管理转型深化客户资产配置变革，政策支持下的并购整合则有望重塑竞争格局。

### 对交易影响可能最大的10个信息

1. **A股再融资规模同比增超三倍，达1.02万亿元，显著提振券商投行业务收入**。  
2. **主板再融资受理单数占51%，传统行业龙头融资需求旺盛，带动头部券商项目储备增长**。  
3. **2024年前三季度42家可比券商投行业务净收入同比增长23.46%，行业复苏趋势明确**。  
4. **中信证券、华泰联合等前五家券商占据再融资承销收入近70%，行业集中度持续提升**。  
5. **中国财富管理总规模突破170万亿元，居民可投资资产超300万亿元，为券商财富管理业务提供巨大增长空间**。  
6. **2024年10月至2025年9月平均月新开户数达290万户，较2023年增长48%，经纪业务活跃度显著回升**。  
7. **公募基金规模达36.74万亿元，创历史新高，券商代销与资管业务持续受益**。  
8. **中金公司筹划吸收合并信达证券、东兴证券，若完成将跃居行业净资产第四、净利润第六**。  
9. **合并后中金公司归母净资产将增至1727亿元，资本实力大幅提升，增强衍生品与自营业务空间**。  
10. **监管明确支持打造10家左右优质头部券商，推动形成2-3家具备国际竞争力的投行集团**。

## 详细部分

### 1. 事件或主题概述

近期券商行业呈现多重积极信号：**A股再融资市场迎来量价齐升，2024年前11个月再融资规模达1.02万亿元，同比增长约368%**，成为投行业务增长的核心引擎。与此同时，**居民“存款搬家”趋势加速，财富管理规模突破170万亿元，权益市场新开户数大幅增长，为券商经纪与财富管理业务注入新动能**。此外，**中金公司筹划重大资产重组，拟换股吸收合并信达证券与东兴证券，标志着券商行业并购整合进入实质性推进阶段**。

政策层面，“新国九条”及证监会关于建设一流投资银行的指导意见明确提出支持头部机构通过并购重组提升竞争力，为行业集中度提升提供制度基础。中央汇金旗下多券商牌照布局也为整合创造条件。

### 2. 核心驱动与影响分析：投资逻辑分步解释

#### 驱动一：再融资市场复苏，投行业务迎来量利双升
- **量的提升**：2024年再融资项目数量与规模均大幅增长，主板项目占主导，平均融资规模超15亿元，体现大市值企业融资需求强劲。
- **利的改善**：头部券商凭借项目获取能力与执行经验，在再融资承销市场中占据主导地位，前五家机构收入占比近70%，马太效应加剧。
- **逻辑链条**：企业融资需求释放 → 券商投行业务量增 → 头部券商受益更明显 → 盈利质量改善 → 估值修复。

#### 驱动二：财富管理转型深化，从通道向资产配置升级
- **结构性变化**：券商经纪业务正从传统交易佣金模式向管理费模式转型，财富管理成为新的增长极。
- **客户基础扩大**：新开户数增长48%，两融余额达2.5万亿元新高，显示居民权益配置意愿上升。
- **逻辑链条**：存款利率下行 → 居民资产配置转向权益 → 券商客户流量增长 → 财富管理规模扩张 → 中收占比提升。

#### 驱动三：行业整合预期强化，头部券商竞争力有望跃升
- **中金合并案例**：若顺利完成对信达证券、东兴证券的吸收合并，中金公司将实现净资产与净利润规模跃升，并获得地理网络互补。
- **资本实力增强**：合并后自营权益类证券/净资本比例将优化，为衍生品、跨境业务提供更大空间。
- **逻辑链条**：政策支持并购 → 头部券商整合加速 → 规模效应显现 → ROE稳定性提升 → 估值中枢上移。

### 4. 主要风险与不确定性

1. **市场交易量回落风险**：若股市交投活跃度下降，将直接影响券商经纪、两融与自营收益。
2. **权益市场大幅调整**：资本市场波动加剧可能导致再融资项目推迟或取消，影响投行业绩连续性。
3. **监管政策收紧**：金融监管趋严可能限制创新业务发展，特别是衍生品与表外业务。
4. **并购整合不确定性**：中金公司合并案涉及A+H股结构与多方利益协调，存在审批与 execution 风险。
5. **利率环境变化**：若存款利率反弹或理财收益率下行放缓，可能减缓“存款搬家”节奏。
6. **行业竞争加剧**：银行、基金、互联网平台在财富管理领域竞争激烈，券商面临客户争夺压力。
